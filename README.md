### SecXss

> 被动xss扫描器，得益于**XSStrike**与**MitmProxy**开源项目的强大支持！



### Thanks

代理模块源于**MitmProxy**提供的支持。

```
https://github.com/mitmproxy/mitmproxy
```

核心的检测引擎源于**XSStrike**提供的支持。

```
https://github.com/s0md3v/XSStrike
```



### Statement

旨在协助安全测试人员更加及时准确发现Xss威胁，未经授权的安全测试一律禁止！

本项目完全开源!



### Start

以代理模式启动

```
python secxss.py
```
